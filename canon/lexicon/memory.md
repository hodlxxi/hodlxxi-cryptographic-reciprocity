# Memory

## Definition

Memory is retained evidence of past behavior that can shape future interpretation, prediction, and response.

## Why it matters

CRT treats memory as the substrate of reputation, identity continuity, obligation, forgiveness, and long-running trust.

## Observation

Memory appears when past conduct changes how present action is interpreted.

## Counterexamples

Storage is not memory unless it changes interpretation or future behavior.

A complete log is not necessarily useful memory.

Immutable memory without forgiveness can create brittle systems.

## Historical analogies

Oral tradition, family history, public records, credit history, court precedent, merchant books, blockchains, and signed receipts all preserve different forms of memory.

## Implications for HODLXXI

HODLXXI should preserve enough memory for accountability while avoiding meaningless accumulation, reputation laundering, and irreversible punishment for repairable failures.

## Open questions

What should be remembered forever?

What should decay?

What should remain private?

What should be forgiven but not forgotten?
